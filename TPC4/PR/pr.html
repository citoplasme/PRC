Abstract

Este trabalho semanal baseia-se na criação de uma aplicação web capaz de dar respostas sobre um repositório GraphDB. Desta forma, pretende-se que a partir da listagem das obras na ontologia, o utilizador possa aceder às partituras de uma obra em específico.

Assim, a primeira rota deve efetuar uma query ao repositório, obtendo-se os dados da obra (título, número de partituras e identificador). No segundo caso, tendo como base uma obra e através do identificador da mesma, devem ser apresentadas as partituras, bem como os seus atributos. Note-se que estes são opcionais, pelo que esses dados devem ser tidos em conta tanto na query, como na apresentação dos mesmos.
