Abstract

Este trabalho semanal baseia-se na criação de uma aplicação web capaz de dar respostas sobre repositórios GraphDB. Desta forma, pretendia-se que existisse uma forma de selecionar entre os repositórios possuídos, bem como escrever a query a executar. Por fim, o utilizador deveria ter a possibilidade de limpar a área de texto com as respostas obtidas.

Tanto a área textual para a inserção da query, como a área dos resultados, tratam-se de text areas, tendo placeholders e ações a si associadas por um botão. No lado esquerdo do ecrã, o utilizador pode selecionar o repositório a interrogar dentro dos seus (valores carregados com a página), bem como indicar a query por via textual. Ao selecionar o botão de execução, é gerada uma ação com auxílio de AJAX para executar e alocar os dados da resposta na área respetiva. Note-se que as duas áreas textuais são expansíveis, sendo que os resultados e as queries possuem dimensões elevadas por norma. Além disso, de modo a limpar a área de respostas, existe outra ação AJAX associada a um botão, sendo que atribui o valor nulo ao conteúdo daquele componente.
