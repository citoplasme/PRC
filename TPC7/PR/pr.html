Abstract

Este trabalho semanal baseia-se na execução de interrogações SPARQL para povoar uma ontologia sobre o mundo do cinema.

O primeiro passo foi a execução das interrogações, com consequente parsing do conteúdo das mesmas, formando-se vários datasets, um para cada classe da ontologia. Após a obtenção destes conjuntos de dados, a ideia passou por converter, um a um, os registos para o formato ontológico. Para isto, teve-se como base os indivíduos criados durante a última aula.

Assim, foram gerados 3 ficheiros turtle, sendo estes referentes a output, que depois seriam fundidos com as declarações iniciais para finalizar a ontologia. Note-se que devido à irregularidade dos dados vindos da DBPedia, existem registos com identificadores duplicados. Além disso, devido à definição inicial da ontologia, dados como os escritores e músicos não foram utilizados por faltarem dados de obras. Também é de destacar que não foram associados os atores aos respetivos filmes, mas bastaria encontrar uma interrogação na DBPedia para relacionar filmes com atores.
