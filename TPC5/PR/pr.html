Abstract

Este trabalho semanal baseia-se no desenvolvimento de uma interrogação SPARQL sobre a DBPedia, de modo a gerar um dataset sobre nadadores. Este deve permitir informação incompleta, ou seja, registos sem todos os campos preenchidos. Os campos pretendidos a apresentar no dataset são: foaf:name, foaf:gender, dbo:birthDate, dbo:birthPlace (Opcional), dbo:deathDate (Opcional) e dbo:abstract@en (Opcional).
